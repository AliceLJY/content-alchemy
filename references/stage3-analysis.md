# Stage 3: Deep Analysis & Truth Check

> Read this when entering Stage 3. Contains 5-dimension analysis instructions, Truth Table format, decision rules, and why human verification matters.

---

## Action

Perform 5-dimension analysis on extracted sources.

## Source Truth Table Format

| Core Claim | Original Source | Verification Method | Status | Human Judgment |
|-----------|----------------|-------------------|--------|---------------|
| "Llama 4 has 405B params" | YouTube/xxx 12:34 | Video subtitle verbatim | Verified | Credible |
| "Meta internal tests beat GPT-4" | Comment speculation | No primary source | Secondary | Delete |
| "Expected release 2025 Q2" | Bilibili description | UP host paraphrase | Unofficial | Rephrase to "reportedly" |

## Decision Rules

- **Keep**: Verifiable primary source
- **Rephrase**: Secondary source (add "according to XX")
- **Delete**: No source / AI speculation

## Why Human Verification Matters

AI cannot judge source credibility. Only humans can decide:
- Is this an official announcement or rumor?
- Is the source biased?
- Should we include this unverified claim?

**User's role** -- verify each claim:
1. **Is this from the original content?** (Not comments/descriptions)
2. **Can you locate the exact timestamp/paragraph?** (Not "approximately mentioned")
3. **Is the source authoritative?** (Official > Secondary > Speculation)

## Checkpoint

Present **Source Truth Table** (Core Claims vs. Real Sources). **User must confirm accuracy before writing.**
